It's alpha+!

Which means that there might be bugs and might even not be playable on your system/config. But for most people (±90%) it runs well. Ask around on Discord if you run into problems.

Minimum Requirements

CPU: Dual core 2.7ghz CPU & newer than 2010
GPU: GTX950 and up
RAM: 8gb and up
Requirements may change due to upgrades we are working on.

‍

1. Download the launcher

First download the installer.

Download Windows Installer
.exe
Download Linux launcher
.AppImage
Download Linux Installer
Flatpak from Flathub
For Windows we recommend installing in the default install directory. On many systems updates only go to the default install folder.

During installation on Windows please make sure your final destination folder is named Beyond-All-Reason, otherwise you might have some trouble with updating.

‍

2. Run the launcher

Run the launcher (and if prompted, allow it to make connections), and click the big blue ''Update" button to start the update proces. This will download all latest assets and can take a while. The launcher will also restart if it got updates as well.

PS. We update the game multiple times per day. Eventually we will reduce this to once or twice a week, or for hot-fixes.

3. Run the game

After successful updates you can click ''Start". On your first launch you can register a new player account. Please fill in the credentials. E-mail needs to be entered, but we currently don't use it for (double) verification or other things. Do remember your password!

‍

Optionally

Download a map-pack

Currently there's only a few maps included in the download. You can download our full map-pack, we also use on our servers. Unpack/download all map files (mapnames.sd7) into your game-folder in ../Beyond-All-Reason/data/maps/.

Reinstall

Do a proper uninstall first

If you need to reinstall BAR (on Windows) - use the appropriate uninstall procedure through "add or remove programs" in the windows settings system. A new install will then work properly.
